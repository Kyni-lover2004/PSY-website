# 🚀 Как запустить проект

## 1️⃣ Запуск бэкенда

```bash
cd C:\freelance_projects\backend
py main.py
```

Бэкенд запустится на: **http://localhost:8000**

---

## 2️⃣ Запуск фронтенда

```bash
cd C:\freelance_projects\frontend
npm run dev
```

Фронтенд запустится на: **http://localhost:3000**

---

## 🔐 Вход для админа

1. Откройте **http://localhost:3000/login**
2. Введите:
   - Логин: `admin`
   - Пароль: `admin123`
3. Нажмите "Войти"
4. Автоматически попадёте в админ-панель: **http://localhost:3000/admin**

---

## 📋 Структура

| Компонент | Адрес | Назначение |
|-----------|-------|------------|
| **Бэкенд API** | localhost:8000 | Все данные и логика |
| **Фронтенд** | localhost:3000 | Сайт для пользователей |
| **Админка** | localhost:3000/admin | Панель администратора |

---

## 🔒 Безопасность

- Админка доступна только пользователям с `role === "admin"`
- Обычные пользователи автоматически перенаправляются на `/dashboard`
- Неавторизованные пользователи перенаправляются на `/login`
- Все admin API endpoints проверяют роль пользователя
