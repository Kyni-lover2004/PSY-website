# 🚀 КАК ЗАПУСТИТЬ ПРОЕКТ

## Быстрый старт

### 0. Инициализация базы данных (ПЕРВЫЙ ЗАПУСК)
```
cd <путь к папке проекта>/backend
py migrate_db.py
```
⚠️ **Обязательно выполните этот шаг при первом запуске или после обновлений с GitHub!**

> 💡 **Примеры путей:**
> - `C:\freelance_projects\backend`
> - `C:\Users\ADMIN\Music\PSY-website\backend`
> 
> Укажите путь туда, где у вас лежит папка `backend`

### 1. Запуск бэкенда
```
cd <путь к папке проекта>/backend
py main.py
```
Бэкенд: **http://localhost:8000**

### 2. Запуск фронтенда (новое окно)
```
cd <путь к папке проекта>/frontend
npm install
npm run dev
```
Фронтенд: **http://localhost:3000**

### 3. Вход в админку
- Логин: `admin`
- Пароль: `admin123`
- URL: http://localhost:3000/login → автоматический редирект в /admin

## Если что-то не работает

### Бэкенд не запускается:
```
cd <путь к папке проекта>/backend
py -m py_compile main.py  # проверка синтаксиса
```

### Пересоздать БД:
```
cd <путь к папке проекта>/backend
py migrate_db.py
```

### Фронтенд не запускается:
```
cd <путь к папке проекта>/frontend
npm install
npm run dev
```

### Очистить кэш браузера:
F12 → Application → Local Storage → Clear all
