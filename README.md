# Psycho Archetypes - Тест на архетипы взаимоотношений

Веб-приложение для психологического тестирования "Естественные архетипы взаимоотношений".

## Стек технологий

**Backend:**
- FastAPI (Python)
- SQLAlchemy (SQLite)
- JWT аутентификация

**Frontend:**
- React 18
- React Router
- Recharts (визуализация)
- Tailwind CSS
- Vite

## Установка

### Backend

```bash
cd backend
pip install -r requirements.txt
python main.py
```

Сервер запустится на `http://localhost:8000`

### Frontend

```bash
cd frontend
npm install
npm run dev
```

Приложение запустится на `http://localhost:3000`

## API Документация

API доступно по адресу `http://localhost:8000/docs` (Swagger UI)

### Основные эндпоинты:

- `GET /api/questions/{gender}` - Получить вопросы для пола
- `POST /api/test/complete` - Завершить тест
- `POST /api/compatibility/check` - Проверить совместимость
- `GET /api/profile/{code}` - Получить профиль по коду
- `POST /api/auth/register` - Регистрация
- `POST /api/auth/login` - Вход

## Админ-панель

Доступна по адресу `/admin`

Функционал:
- Редактирование вопросов (особенно для мужчин)
- Редактирование описаний архетипов
- Просмотр заявок на консультацию

## Алгоритм совместимости

Индекс комплементарности А/П:
- **A** — количество активных совпадений
- **П** — количество пассивных конфликтов

Интерпретация:
- `7/0` — Максимальная прочность
- `0/7` — Минимальная прочность
- `> 1` — База есть, но есть очаги конфликтов
- `< 1` — Союз временный
- `A ≥ 3` — Отношения сохраняются
