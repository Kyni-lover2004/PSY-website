# Psycho Archetypes

Веб-приложение для психологического тестирования "Естественные архетипы взаимоотношений". Определяет архетип личности в отношениях и рассчитывает совместимость между партнёрами.

## Установка и запуск

### Требования
- Python 3.10+
- Node.js 18+
- npm или yarn

### Backend

```bash
cd backend
pip install -r requirements.txt
python main.py
```

API: http://localhost:8000  
Swagger UI: http://localhost:8000/docs

### Frontend

```bash
cd frontend
npm install
npm run dev
```

Приложение: http://localhost:5173

## API

Базовый URL: `/api`

### Аутентификация

**POST /api/auth/register**

```json
{
  "login": "user123",
  "password": "securepass",
  "gender": "female",
  "orientation": "hetero"
}
```

**POST /api/auth/login**

```json
{
  "login": "user123",
  "password": "securepass"
}
```

### Тестирование

**GET /api/questions/{gender}** — gender: `male` | `female`

**POST /api/test/complete**

```json
{
  "session_id": "sess_12345",
  "answers": [
    { "question_id": 1, "value": true },
    { "question_id": 2, "value": false }
  ],
  "gender": "female"
}
```

### Совместимость

**GET /api/profile/{compatibility_code}**

**POST /api/compatibility/check**

```json
{
  "code1": "PSY-20260401-ABC123",
  "code2": "PSY-20260401-XYZ789"
}
```

## Алгоритм совместимости

- **A** (Active) — количество активных совпадений архетипов
- **П** (Passive) — количество пассивных конфликтов

Интерпретация индекса A/П:

| Индекс | Значение |
|--------|----------|
| 7/0 | Максимальная прочность союза |
| 0/7 | Минимальная прочность |
| A > П | База есть, но есть очаги конфликтов |
| A < П | Союз временный |
| A ≥ 3 | Отношения имеют тенденцию к сохранению |
