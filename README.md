# Psycho Archetypes - Тест на архетипы взаимоотношений

Веб-приложение для психологического тестирования "Естественные архетипы взаимоотношений".

## 🌸 Описание

Приложение помогает определить архетип личности в отношениях и рассчитать совместимость между партнёрами на основе психологического теста.

## 🚀 Установка и запуск

### Требования
- Python 3.10+
- Node.js 18+
- npm или yarn

### Backend

```bash
cd backend

# Установка зависимостей
pip install -r requirements.txt

# Запуск сервера (порт 8000)
python main.py
```

**API доступно:** http://localhost:8000  
**Swagger UI:** http://localhost:8000/docs

### Frontend

```bash
cd frontend

# Установка зависимостей (если не установлены)
npm install

# Запуск dev-сервера (порт 3000)
npm run dev
```

**Приложение доступно:** http://localhost:3000

## 📡 API Документация

### Базовый URL
```
http://localhost:8000/api
```

### Аутентификация

#### Регистрация
```http
POST /api/auth/register
Content-Type: application/json

{
  "surname": "Иванова",
  "name": "Мария",
  "phone": "+79991234567",
  "password": "пароль123",
  "gender": "female"
}
```

**Ответ:**
```json
{
  "user_id": 1,
  "name": "Мария",
  "surname": "Иванова",
  "gender": "female",
  "compatibility_code": "PSY-20260401-ABC12345"
}
```

#### Вход
```http
POST /api/auth/login
Content-Type: application/json

{
  "phone": "+79991234567",
  "password": "пароль123"
}
```

**Ответ:**
```json
{
  "user": {
    "id": 1,
    "name": "Мария",
    "surname": "Иванова",
    "phone": "+79991234567",
    "gender": "female"
  },
  "compatibility_code": "PSY-20260401-ABC12345"
}
```

### Тестирование

#### Получить вопросы
```http
GET /api/questions/{gender}
# gender: "male" или "female"
```

#### Завершить тест
```http
POST /api/test/complete
Content-Type: application/json

{
  "session_id": "uuid",
  "answers": [
    {"question_id": 1, "value": true},
    {"question_id": 2, "value": false}
  ],
  "gender": "female",
  "name": "Мария"
}
```

### Профиль и совместимость

#### Получить профиль
```http
GET /api/profile/{code}
```

#### Проверить совместимость
```http
POST /api/compatibility/check
Content-Type: application/json

{
  "code1": "PSY-20260401-ABC12345",
  "code2": "PSY-20260401-XYZ67890"
}
```

## 💕 Алгоритм совместимости

### Индекс комплементарности А/П

- **A** — количество активных совпадений
- **П** — количество пассивных конфликтов

### Интерпретация

| Индекс | Значение |
|--------|----------|
| `7/0` | Максимальная прочность союза |
| `0/7` | Минимальная прочность (развал) |
| `> 1` | База есть, но есть очаги конфликтов |
| `< 1` | Союз временный (против больше чем за) |
| `A ≥ 3` | Отношения имеют тенденцию к сохранению |

### Статусы архетипов

| Статус | Баллы | Описание |
|--------|-------|----------|
| **A** (Active) | ≥ 4 | Активный |
| **M** (Middle) | 3 | Средний |
| **N** (Neutral) | 2 | Неопределенный |
| **P** (Passive) | ≤ 1 | Пассивный |
